## 一、分钟上手

### 打开工具

双击 `track_builder.html`，浏览器打开即可使用。

### 铺设一条最简单赛道（5 步）

```
步骤 1：选工具          步骤 2：定方向与大小        步骤 3：点击放置
┌────────────┐         ┌──────────────────┐        ┌──────────────────┐
│ 左侧工具栏   │   →    │ 移动鼠标，半透明    │   →   │ 点击鼠标左键，    │
│ 已选「直道」 │        │ 预览跟随光标       │        │ 赛道段落下         │
└────────────┘         └──────────────────┘        └──────────────────┘
```

**完整流程：**

1. 打开页面 → 默认已选中「直道」
2. 鼠标移到画布中央 → 看到半透明直道跟随光标
3. 左键点击 → 直道落下，**末端自动成为下一段的起点**
4. 按 `2` 切换到「弯道」→ 移动鼠标定角度 → 点击放置
5. 继续铺弯、直道……铺完后按 `Ctrl+S` 保存 YAML

|------|------|

## 四、赛道段工具详解

### ▭ 直道（快捷键 `1`）

最基础的赛道段。起点到鼠标的距离 = 长度，铺设方向 = 起点到鼠标的方向（开启吸附时对齐 15° 整数倍）。

**参数**：长度（米）

**规则限制**：
- 夹在两个发夹弯之间时，长度 **≤ 60 m**
- 一端为大弯道（≥60° 且 R≥15m）时，长度 **≤ 45 m**

### ◠ 弯道（快捷键 `2`）

圆弧段。铺设方向 = 起点到鼠标的方向，终点 = 鼠标位置；半径由滚轮 / `Q` / `E` 调整。

**参数**：
- 半径：弯道大小（滚轮铺设时调节，或属性面板输入）
- 角度：转弯程度（5°~270°）
- 方向：左转 / 右转（按 `R` 翻转，或属性面板选）

**小技巧**：按 `Q` / `E` 微调半径 ±1 米。

### ↩ 发夹弯（快捷键 `3`）

180° 急转弯，是弯道的特例。用于掉头。

**规则限制**：半径 **≥ 9 米**（外径最小 9 米，否则标红警告）

### ⋮ 蛇形穿桩（快捷键 `4`）

等间距排列的桩桶，车辆需左右绕行。铺设方向 = 起点到鼠标方向，桩数由距离自动推算。

**参数**：
- 桩数：2~20 桩
- 间距：相邻桩的间隔距离

**规则限制**：间距 **7.62 米 ~ 12.19 米**

### ∿ 复合弯（快捷键 `5`）

多个半径递减的弯道串联（类似「减速弯」）。一次性铺入一整串。

**参数**：半径序列（如 `18, 12, 8`）、单段角度、方向

### ▲ 起点

点击画布移动起点位姿。起点位置 = 新车起步位置。

## 五、编辑已铺设段（拖拽控制点）

选中段后，**画段中心线**上会出现控制点，拖拽即可实时编辑：

| 控制点 | 颜色 | 拖拽效果 |
|--------|------|----------|
| 起点 | 🟢 绿 | 固定（起点位姿由「▲ 起点」工具移动） |
| 终点 | 🔴 红 | 直道→改长度 / 弯道→改角度（终点沿圆弧滑动跟手） |
| 弧段中点 | 🟠 橙 | 改弯道半径（仅弯道/发夹可见） |

**拖拽后后续段自动跟进**，保持赛道连续。

### 选中段

在画布上**点击**已铺设的段即可选中：
- 段高亮变蓝
- 出现控制点
- 右侧面板显示该段参数

### 精确修改参数

在右侧属性面板直接输入数值（数字框精确输入，拖动/点击即生效，实时预览）。

## 六、删除锥筒

点击工具栏 **✕ 删筒** 进入删筒模式。

### 操作方式

1. 点击 **✕ 删筒** 按钮（高亮 = 已进入模式）
2. 移动鼠标 → 最近的锥筒被**红环高亮**（点击前确认目标）
3. 左键点击 → **删除该单个锥筒**
4. 可连续删除多个锥筒
5. 按 `1`–`5` 选其它工具、或按 `Esc`，退出删筒模式

### 删除规则

- **蓝/黄锥**（赛道边界、蛇形桩）：点击哪个删哪个，单锥粒度
  - 连续点击同一站点的蓝锥和黄锥 → 清空该站点
- **橙锥**（起跑线）：单锥删除
- 删除可撤销（`Ctrl+Z` / `Esc`）

### 与其它操作的关系

删筒模式是**独立模式**：进入后左键**只删锥筒**，不会铺新段、不会选中段、不会闭合赛道。

## 七、规则校验（自动实时检查）

铺设/编辑时**自动检查**赛规合规性：

### 合规

- 赛道段显示为**蓝/绿色实线**
- 状态栏显示绿色 ✓

### 违规

- 赛道段变为**红色并闪烁**
- **状态栏显示具体原因**，例如：

```
⚠ 直道长度 65.0m > 60m（两端发夹弯间上限）
⚠ 发夹弯半径 7.0m < 9m 最小外径
⚠ 蛇形间距 6.0m < 7.62m 下限
```

### 五条赛规一览

| 规则 | 条件 | 限制 |
|------|------|------|
| 直道-发夹间 | 前后都是发夹弯 | ≤ 60 m |
| 直道-大弯道间 | 一端为大半径弯道（≥60° 且 R≥15m） | ≤ 45 m |
| 发夹弯外径 | 任何发夹弯 | ≥ 9 m |
| 蛇形桩间距 | 蛇形穿桩段 | 7.62 ~ 12.19 m |
| 赛道宽度 | 全局 | 3.5 m（锥筒对间距） |

## 八、删除整段与撤销

### 删除整段

选中段后按 `Del` / `Backspace`，或点面板底部「删除本段」按钮。

### 撤销

- `Ctrl+Z` 或 `Esc`：撤销上一段铺设、编辑、删筒、移动起点等所有操作
- 撤销栈深度 50 步

## 九、导出与预览

### 👁 预览（顶栏按钮 / `Ctrl+P`）

弹窗显示整条赛道的俯瞰图，包含所有锥筒位置。无需切到其他工具即可确认效果。

### 💾 保存 YAML（顶栏按钮 / `Ctrl+S`）

下载一个 `.yaml` 文件，格式与现有赛道文件（`trackdrive.yaml`、`autocross_high_speed.yaml`）**完全一致**。

**导出的 YAML 包含**：
- `track.type`：赛道类型
- `start_pose`：起点位姿（x, y, yaw）
- `blue_cones`：左边界锥筒坐标
- `yellow_cones`：右边界锥筒坐标
- `orange_cones`：起/终点线锥筒坐标

## 十、画布操作

| 操作 | 方法 |
|------|------|
| **缩放** | 滚轮（以鼠标位置为中心放大/缩小） |
| **平移** | 按住右键或中键拖拽 |
| **适应视图** | 按 `F` 或点工具栏「适应视图」—— 让全部赛道完整显示在画布中 |
| **角度吸附** | 按 `G` 开关（铺设时自动吸附到 15° 整数倍，方便铺直角/直线） |

## 十一、快捷键速查表

| 快捷键 | 功能 |
|--------|------|
| `1` `2` `3` `4` `5` | 切换段类型：直道 / 弯道 / 发夹弯 / 蛇形 / 复合弯 |
| `R` | 旋转铺设方向 90° / 弯道左右翻转 |
| `Q` `E` | 弯道半径 −1m / +1m |
| 滚轮 | 画布缩放 / 铺设弧段时调半径 |
| 左键点击 | 铺设段 / 选中段 / 删筒模式下删除锥筒 |
| 拖拽控制点 | 编辑已铺设段的参数（长度/角度/半径） |
| 右键/中键拖拽 | 平移画布 |
| `Ctrl+Z` / `Esc` | 撤销上一段（含删筒） |
| `Del` / `Backspace` | 删除选中段 |
| `G` | 切换网格吸附（15° 整数倍） |
| `F` | 适应视图（全赛道可见） |
| `Ctrl+S` | 导出 YAML |
| `Ctrl+P` | 预览 |
| `H` | 显示/隐藏本帮助 |

## 十三、常见问题

### Q：铺设时方向不对，想转 90°？

**A**：按 `R` 键旋转铺设方向。按 `G` 开启角度吸附，自动对齐 15° 整数倍。

### Q：某段变红了怎么办？

**A**：看**底部状态栏**，会显示具体违规原因（如「半径 7m < 9m」），修改对应参数即可恢复绿色。

### Q：铺错了怎么撤回？

**A**：`Ctrl+Z` 或 `Esc` 撤销上一段。也可选中某段按 `Del` 删除。

### Q：铺好的赛道能修改某一段吗？

**A**：点击该段选中，拖拽控制点或在右侧面板修改数值，后续段自动跟进。

### Q：导出的 YAML 能被模拟器使用吗？

**A**：可以。格式完全对齐 `trackdrive.yaml` 等现有文件，可直接喂给 `visualize_track.py` 和模拟器。

### Q：赛道可以保存为草稿继续编辑吗？

**A**：目前每次导出会覆盖 YAML。建议导出时用不同文件名保存版本（文件名含时间戳，不会重名）。

### Q：想删除某个多余的锥筒怎么办？

**A**：点工具栏 **✕ 删筒**，移动鼠标到锥筒上（红环高亮），左键点击删除。详见[六、删除锥筒](#六删除锥筒)。

### Q：删筒想一次删多个？

**A**：进入删筒模式后，可连续点击多个锥筒逐个删除，无需重复点工具栏按钮。

### Q：删错了能恢复吗？

**A**：能。`Ctrl+Z` 或 `Esc` 可撤销删筒操作（撤销栈 50 步）。
