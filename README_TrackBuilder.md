# WUTA-FSD Track Builder — 使用指南

> 像「都市天际线」铺路一样，用鼠标和键盘铺设 Formula Student 赛道。

---

## 一、一分钟上手

### 打开工具

双击 `track_builder.html`，浏览器打开即可使用。**无需安装任何软件。**

### 铺设你的第一条赛道（3 步）

```
步骤 1：选工具          步骤 2：定方向与大小        步骤 3：点击放置
┌────────────┐         ┌──────────────────┐        ┌──────────────────┐
│ 左侧工具栏   │   →    │ 移动鼠标，半透明    │   →   │ 点击鼠标左键，    │
│ 点击「直道」 │        │ 预览跟随光标       │        │ 赛道段落下         │
└────────────┘         └──────────────────┘        └──────────────────┘
```

**完整流程举例**（铺设一条简单赛道）：

1. 打开页面 → 默认已选中「直道」工具
2. 鼠标移到画布中央 → 看到半透明直道跟随光标
3. 左键点击 → 直道落下，**末端自动成为下一段的起点**
4. 按 `2` 切换到「弯道」→ 移动鼠标定角度 → 点击放置
5. 按 `3` 切换到「发夹弯」→ 点击放置
6. 按 `4` 切换到「蛇形穿桩」→ 点击放置
7. 点右上角「保存 YAML」→ 完成！

## 三、赛道段类型详解

### ▭ 直道（快捷键 `1`）

最基础的赛道段。鼠标距离起点越远，直道越长。

```
        点击
          ↓
   ★━━━━━━━━━━━━━━━━━━━→
   起点        鼠标位置 = 长度
```

**参数**：长度（米）

**规则限制**：
- 夹在两个发夹弯之间时，长度 **≤ 60 米**
- 一端为大弯道时，长度 **≤ 45 米**

---

### ◠ 弯道（快捷键 `2`）

圆弧段，可定半径、角度、左转/右转。

**参数**：
- 半径：弯道大小（拖动控制点或属性面板输入）
- 角度：转弯程度（5°~270°）
- 方向：左转 / 右转

**小技巧**：按 `Q` / `E` 微调半径 ±1 米，按 `R` 快速切换左右转。

---

### ↩ 发夹弯（快捷键 `3`）

180° 急转弯，是弯道的特例。用于掉头。

```
           ╭━━━╮
   ★━━━━━━━┛   ┗━━━━━━━→
   起点      ↑       出口（方向相反）
          掉头
```

**规则限制**：半径 **≥ 9 米**（外径最小 9 米，否则标红警告）

---

### ⋮ 蛇形穿桩（快捷键 `4`）

等间距排列的桩桶，车辆需左右绕行。

```
   ★━━●━━  ━━●━━  ━━●━━  ━━●━━  ━━●→
     左 右   左 右   左 右   左 右
```

**参数**：
- 桩数：2~20 桩
- 间距：相邻桩的间隔距离

**规则限制**：间距 **7.62 米 ~ 12.19 米**

---

### ∿ 复合弯（快捷键 `5`）

多个半径递减/连续的弯道串联（类似「减速弯」「半径渐小弯」）。

```
   ╭━━╮╭━╮╭╮
   ┃  ┃┃ ┃┃┃  半径：18 → 12 → 8（递减）
   ╰━━╯╰━╯╰╯
```

**参数**：半径序列（如 `18, 12, 8`）、角度、方向

## 四、编辑已铺设段

### 选中段

在画布上**点击**已铺设的段即可选中：
- 段高亮变蓝
- 出现控制点：🟢 起点、🔴 终点、🟠 弧段半径控制点
- 右侧面板显示该段参数

### 拖拽控制点

| 控制点 | 拖拽效果 |
|--------|---------|
| 🔴 终点 | 直道→改长度 / 弯道→改角度 |
| 🟠 弧中点 | 改弯道半径 |

**拖拽后后续段自动跟进**，保持赛道连续。

### 精确修改参数

在右侧属性面板直接输入数值：
- 数字框：精确输入
- 拖动/点击即生效，实时预览

### 删除选中段

按 `Delete` 或 `Backspace`，或点面板底部「删除本段」按钮。

## 五、画布操作

| 操作 | 方法 |
|------|------|
| **缩放** | 滚轮（以鼠标位置为中心放大/缩小） |
| **平移** | 按住右键或中键拖拽 |
| **适应视图** | 按 `F` 或点工具栏「适应视图」—— 让全部赛道完整显示在画布中 |
| **角度吸附** | 按 `G` 开关（铺设时自动吸附到 15° 整数倍，方便铺直角/直线） |

## 六、规则校验（自动实时检查）

铺设/编辑时**自动检查**赛规合规性：

### 合规

- 赛道段显示为**蓝/绿色实线**
- 状态栏显示绿色 ✓

### 违规

- 赛道段变为**红色并闪烁**
- **状态栏显示具体原因**，例如：

```
⚠ 直道长度 65.0m > 60m（两端发夹弯间上限）
⚠ 发夹弯半径 7.0m < 9m 最小外径
⚠ 蛇形间距 6.00m < 7.62m 下限
```

### 五条赛规一览

| 规则 | 条件 | 限制 |
|------|------|------|
| 直道-发夹间 | 前后都是发夹弯 | ≤ 60 m |
| 直道-大弯道间 | 一端为大半径弯道 | ≤ 45 m |
| 发夹弯外径 | 任何发夹弯 | ≥ 9 m |
| 蛇形桩间距 | 蛇形穿桩段 | 7.62 ~ 12.19 m |
| 赛道宽度 | 全局设定 | ≥ 3.5 m（默认 3.5m） |

## 七、导出与预览

### 👁 预览（左上角按钮 / `Ctrl+P`）

弹窗显示整条赛道的俯瞰图，包含所有锥筒位置。无需切到其他工具即可确认效果。

### 💾 保存 YAML（左上角按钮 / `Ctrl+S`）

下载一个 `.yaml` 文件，格式与现有赛道文件（`trackdrive.yaml`、`autocross_high_speed.yaml`）**完全一致**。

**后续使用这个文件**：

```bash
# 用现有可视化工具打开
python visualize_track.py 你保存的文件.yaml

# 带边界线显示
python visualize_track.py --lines 你保存的文件.yaml
```

导出的 YAML 文件包含：
- `track.type`：赛道类型
- `start_pose`：起点位姿（x, y, yaw）
- `blue_cones`：左边界锥筒坐标
- `yellow_cones`：右边界锥筒坐标
- `orange_cones`：起/终点线锥筒坐标


## 十、常见问题

### Q：铺设时方向不对，想转 90°？
**A**：按 `R` 键旋转铺设方向。按 `G` 开启角度吸附，自动对齐 15° 整数倍。

### Q：某段变红了怎么办？
**A**：看**底部状态栏**，会显示具体违规原因（如「半径 7m < 9m」），修改对应参数即可恢复绿色。

### Q：铺错了怎么撤回？
**A**：`Ctrl+Z` 或 `Esc` 撤销上一段。也可选中某段按 `Delete` 删除。

### Q：铺好的赛道能修改某一段吗？
**A**：点击该段选中，拖拽控制点或在右侧面板修改数值，后续段自动跟进。

### Q：导出的 YAML 能被模拟器使用吗？
**A**：可以。格式完全对齐 `trackdrive.yaml` 等现有文件，可直接喂给 `visualize_track.py` 和模拟器。

### Q：赛道可以保存为草稿继续编辑吗？
**A**：目前每次导出会覆盖 YAML。建议导出时用不同文件名保存版本（文件名含时间戳，不会重名）。如需继续编辑，重新铺设或后续版本将支持导入 YAML 继续编辑。

### Q：鼠标滚轮只能缩放不能调参数？
**A**：在铺设模式下，先移动鼠标定好方向和大小（预览会跟随），然后再滚动需要再点一下画布外区域退出铺设——下一步将支持滚轮直接调参。

---

> 🎯 **一句话总结**：选工具 → 鼠标定方向大小 → 点击放置 → 红改绿 → 导出 YAML。就这么简单。
